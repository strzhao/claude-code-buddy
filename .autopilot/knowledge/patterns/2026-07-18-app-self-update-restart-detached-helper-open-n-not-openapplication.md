# app 自更新重启用 detached helper（Process + open -n）而非 NSWorkspace.openApplication

<!-- tags: restart, relaunch, nsworkspace, createsnewapplicationinstance, open-n, detached-process, launchservices, upgrade, auto-update, mainactor, swift6, debugging, launch-items-not-deterministic, root-cause -->
**Scenario**: 升级后 `restartApp()` 需重启自己。旧实现 `NSWorkspace.shared.openApplication(at: bundleURL, configuration:)`（`OpenConfiguration.createsNewApplicationInstance` 默认 `false`）+ 紧接 `NSApplication.shared.terminate(nil)` 同进程执行。`openApplication` 发现 bundle id 已有运行实例（就是自己）→ LaunchServices 走"激活已有"而非"新建"→ 真机日志 `LAUNCH: Asking CSUI to launch 0 items`；紧接 `terminate` 杀唯一实例 → 净结果 0 个运行实例 → **app 消失不再回来**。叠加时序缺陷：openApplication 异步派给 LaunchServices daemon，terminate 同步立即杀进程，daemon 处理前进程已死。注意：即便 `brew upgrade` 对已是最新版 no-op（退出码 0），`terminationHandler` 仍发 `upgradeCompleted` → 仍走 restartApp → 仍崩。
**Lesson**: 自更新重启用 **detached 独立子进程**模式（Sparkle 同款）：`Process()` spawn `/bin/sh -c "<script>"` 后立即 `terminate`，子进程父退出后由 launchd 收养继续执行。脚本三要素：① `trap '' HUP` 兜底 controlling-terminal 场景（Terminal 启动 app 时父 shell 退出 SIGHUP）；② `pgrep -x <CFBundleExecutable>` 轮询等旧实例真正退出（等 reap，非固定 sleep——避免竞态 + 慢机/高负载不够，上限 N×0.1s 兜底防卡死）；③ `open -n "<bundlePath>"` 强制 LaunchServices 新建实例（`-n` 等价 `createsNewApplicationInstance=true`，是修复核心）。抽 `buildScript(bundlePath:) -> String` 纯函数便于单测验三要素；`Bundle.main.bundleURL.path` 注入双引号包裹防空格分裂。`Process.executableURL`（非 deprecated 的 `launchPath`）。
**Lesson (debugging，重要)**: LaunchServices 日志 `LAUNCH: Asking CSUI to launch N items` **不是 app 是否启动的决定因素**——修复后（新实例确实起来）该日志仍显示 `0 items`（新实例启动后自己也会发）。诊断 app 启动/消失问题用**进程实证**（`pgrep -x`/`ps` 新旧 PID 对比 + `buddy ping` + `/tmp/claude-buddy.sock` 存在 + 新实例 `socket listening` 日志），不要把 LaunchServices 内部日志当根因铁证（本次曾因此归因偏差，靠真机复现修正）。
**Evidence**: 真机日志 `LAUNCH: Asking CSUI to launch 0 items` + `Termination complete. Exiting`（2026-07-17 用户复现，v0.39.0 升 0.40.0）；修复后真机验证 P1（OLD_PID 45089 → NEW_PID 45145 新实例）+ P2（socket 重建）+ P3（`buddy ping` ✓）+ P4（open-settings 无 SIGABRT）；`Sources/ClaudeCodeBuddy/Update/RestartHelper.swift` + `Tests/BuddyCoreTests/Update/RestartHelperAcceptanceTests.swift`（9 条断言三要素）；commit 1a0ecc9。
