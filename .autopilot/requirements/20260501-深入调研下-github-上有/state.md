## 设计文档

### 工具调研

**第一梯队（推荐）**：
| 工具 | Stars | 用途 |
|------|-------|------|
| yt-dlp | 160k+ | 下载数千网站音视频，`-x --audio-format mp3` 提取 MP3 |
| you-get | 56.8k | 下载媒体内容，支持 B站/网易云音乐/YouTube |
| ffmpeg | 48k+ | 音频处理，截取高潮片段 |

**第二梯队（特定平台）**：spotDL(18k+, Spotify)、netease-cloud-music-dl(2k+, 网易云)

### 技术方案
- **下载**：yt-dlp + YouTube（最成熟稳定），降级 you-get + B站/网易云
- **截取**：ffmpeg 无损截取（`-ss -t -c copy`）
- **高潮定位**：WebFetch 搜索时间戳 → 降级为默认截取总时长 25%-55%

### 歌曲清单
| 歌曲 | 歌手 |
|------|------|

| 你要跳舞吗 | 新裤子乐队 |
| 春风十里 | 鹿先森乐队 |

### 风险与降级
1. yt-dlp 反爬 → you-get + B站/网易云
2. 搜索匹配不准 → 手动提供 URL
3. 高潮时间戳搜索无结果 → 默认 25%-55% 区间截取
